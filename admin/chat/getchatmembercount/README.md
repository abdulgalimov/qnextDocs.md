# getChatMemberCount

**getChatMemberCount** - получить количество участников в чате.

![](./1.png)

Чаты/Каналы - указываем чат в котором необходимо получить информацию.

Информация доступна в переменной: 
```${chatMembersCountResult}```

```json
"chatMembersCountResult": {
      "ok": true,
      "response": 59,
     }
```
